用eclipse3.3.x里自己带的software updates
升级地址填写http://download.eclipse.org/technology/babel/update-site/europa 就可以了！然后出列表后选择一下简体中文就可以了！
eclipse ->Help ->software updates -> Find and Install ->Search for new features to install ->NEXT->选择New Remote Site 升级地址填写为http://download.eclipse.org/technology/babel/update-site/europa

eclipse 3.4用地址：http://download.eclipse.org/technology/babel/update-site/ganymede

以上内容转自: http://www.eclipseworld.org/bbs/read-cec-tid-17078.html

 

 

 

在以前我收集过善于Eclipse相关的资源下载地址[eclipse所有版本及eclipse中文包下载地址列表收藏] ,如Eclipse各版本下载地址,MyEclipse下载,Ecipse3.2.1中文汉化包下载地址等,但是没能找到3.3.X的中文包,而且也一直相信官方没有中文包,直到今天到网上搜索一下发现很多人都提供Eclipse3.3.X的中文包下载,开始我还以为都是骗人的,直到看到上面那个文章给的是官方地址,我才相信了,这才下载下来试一试.

另外,在上面那个帖子里有人说他的Eclipse是使用的MyEclipse 6.5自带的,使用上面那个办法还是英文的,而没有变成中文.这个问题可以解决,打开..\MyEclipse\eclipse下的eclipse.ini配置文件,把-Duser.language=en 改成-Duser.language=zh,重新启动就OK 了

 

3.3版本的升级地址现在为：
http://download.eclipse.org/technology/babel/update-site/europa
