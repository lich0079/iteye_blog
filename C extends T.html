<C extends T> 是限制 Class<C>里的C的 也就是C需要是T的实现类或者子类

泛型更具体

  1. public static class abc<E extends  CharSequence>{   
  2.         public E e;   
  3.         public abc(E o){   
  4.             this.e=o;   
  5.         }   
  6.         public String toString(){   
  7.             return e.toString();   
  8.         }   
  9.         public E getE(){   
  10.             return e;   
  11.         }   
  12.     }

上面的e可以使CharSequence及其子类。
在class abc中e可以调用CharSequence的方法！
