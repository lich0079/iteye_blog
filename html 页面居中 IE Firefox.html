这个是可以让页面居中的 但开始的时候只有firefox居中了 IE6 8都没有居中

 

后来发现问题出在   <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">

换成下面的IE就可以居中了

<!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitional//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd”>
